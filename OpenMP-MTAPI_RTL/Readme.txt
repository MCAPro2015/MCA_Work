The OpenMP-MTAPI Runtime Library is a stand-alone runtime library that support GNU OpenMP compiler runtime interface.
By linking to the OpenMP Task based Application compiled with GCC compiler, the OpenMP-MTAPI RTL is able to take the OpenMP
tasks and distribute it onto the EMBB MTAPI RTL, to execute on targeted embedded devices.

To Build:
    use the script provided in the folder to generate the libgomp.so.1 shared library
